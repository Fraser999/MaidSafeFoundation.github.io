Events

The Foundation aim to work with schools has been realised with the first project run in collaboration with Catrine Primary

Primary 7 pupils of Catrine Primary have been looking into the First World War for the 100th anniversary of the battle. p>We devised a project to get the children to make line drawing of poppies which were transformed using computer software and the laser cutter and engraved into wood to make lino print blocks.

These were then used to make a prints on fabric of the poppy drawing.

The completed prints were then made into a banner which was hung at their remembrance service.
